# Bucket Sort  
  
 - 가장 간단한 정렬 알고리즘  
 - 처리가 간단한 대신 장소적 자원(배열의 크기)을 많이 소모함  
 - 빈 양동이 배열을 준비하고, 양동이의 번호와 일치하는 모든 데이터를 저장한 다 1번째 양동이의 데이터부터 꺼내는 것  
  
## Time Complexity	 
  
|Situation|Big O|
|:---:|:---:|
|Best|O(n+k)|
|Worst|O(n2)|
|Average|O(n)|
|Space Complexity|O(n+k)|
|Stability|Yes|
