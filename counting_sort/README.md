# Counting Sort (계수 정렬)  
  
 - 원소간 비교하지 않고 각 원소가 몇개 등장하는지 갯수를 세서 정렬하는 방법이다.  
 - 모든 원소는 양의 정수여야 한다.  
 - N <= 10,000,000에서 사용 가능  
 - 시간복잡도는 O(n+k) 로 퀵정렬, 병합정렬에 비해 일반적으로 빠르다.  
 - 정렬을 위한 길이 n의 배열 하나, 계수를 위한 길이 k의 배열 하나. O(n+k) 의 공간복잡도를 가진다.  
  
## 안정정렬  
  
카운팅 정렬은 안정정렬의 형태를 가질 수 있다.  
안정정렬이란 같은 값을 가지는 복수의 원소들이 정렬 후에도 정렬 전과 같은 순서를 가지는 것이다.  
안정정렬이 아닌 대표적인 예는 퀵정렬이다.  
  
누적합을 기반으로 정렬할 때, 누적합 기반의 정렬의 반대인 기존 배열의 가장 큰 인덱스에서 작은 인덱스로 (오른쪽에서 왼쪽으로) 진행해야 안정 정렬이 가능하다.
